# namako.eu - Personal Portfolio & UX Showcase

## Project Overview
My central digital hub, designed to showcase the intersection of psychology-driven UX design and modern frontend engineering.

## Key Features
- **User-Centric Design:** Focused on minimalist aesthetics and intuitive navigation to reduce user friction.
- **Performance Optimization:** High-speed loading achieved through clean code and efficient asset management (relax.js, Swiper).
- **Responsive Architecture:** Fully optimized for all device types, focusing on accessible UI elements.

## Tech Stack
- **Languages:** HTML5, CSS3, JavaScript.
- **Libraries:** jQuery, relax.js, Swiper.
- **Design Philosophy:** Human Factors in IT & Psychology-driven UX.
